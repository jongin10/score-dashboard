# 📚 과목별 성적 추이 분석 대시보드

1학기와 2학기 성적을 비교하여 성적 변화를 추적하고, 성장을 분석해주는 고급 대시보드입니다.

## 🎯 주요 기능

### 1. **1학기 & 2학기 성적 입력**
- 6개 과목별 점수 입력 (0~100점)
- 각 학기별로 분리된 입력 영역
- 직관적이고 간편한 UI

### 2. **성적 변화 시각화**
- **라인 차트**: 1학기와 2학기의 성적 추이를 한눈에 비교
- **비교 차트**: 과목별로 학기 간 점수 변화를 막대 그래프로 표시

### 3. **성적 변화 분석**
- 1학기 평균 vs 2학기 평균
- 평균 상승/하락도 자동 계산
- 최대 상승 과목 표시
- 과목별 점수 변화 상세 비교

### 4. **상세 비교 테이블**
- 모든 과목의 1학기/2학기 성적 한눈에 비교
- 과목별 변화도 (상승 📈 / 하락 📉 / 유지 →)
- 현재 성적 평가 (⭐ 👍 📚 ⚠️)

### 5. **맞춤형 피드백**
- 📊 **전체 성적 변화 분석**: 성장 추세 파악
- ✅ **성적이 오른 과목**: 잘한 과목 확인 및 동기부여
- 📉 **성적이 내려간 과목**: 집중 필요 과목 파악
- ⭐ **현재 강점 과목**: 앞으로도 유지할 과목
- 📚 **집중이 필요한 과목**: 개선 방향 제시
- 🌟 **종합 평가**: 전체 학습 추세 및 조언

## 📖 사용 방법

### Step 1: 성적 입력
1. **"📝 성적 입력"** 탭에서 시작
2. **1학기 성적** 입력 (6개 과목 모두)
3. **2학기 성적** 입력 (6개 과목 모두)
4. **"📊 성적 분석하기"** 버튼 클릭 (또는 Enter 키)

### Step 2: 결과 확인
자동으로 **"📊 성적 분석"** 탭으로 이동하여:
- 라인 차트로 성적 추이 확인
- 비교 차트로 학기 간 차이 시각화
- 통계 정보로 평균 변화 파악
- 비교 테이블로 과목별 상세 비교
- 맞춤형 분석 및 피드백 확인

### Step 3: 초기화
**"🔄 초기화"** 버튼으로 모든 데이터 삭제

## 📊 분석 기준

### 성적 평가
- **90점 이상**: ⭐ 탁월한 성적
- **80~89점**: 👍 우수한 성적
- **70~79점**: 📚 평범한 성적
- **70점 미만**: ⚠️ 개선 필요

### 성적 변화 평가
| 변화도 | 평가 | 피드백 |
|------|------|--------|
| **+5점 이상** | 🚀 눈에 띄는 성장 | 지난 학기 노력이 성과를 냈습니다 |
| **+1~4점** | 👍 성적 향상 | 꾸준한 노력이 결실을 맺고 있습니다 |
| **-4~0점** | → 안정적인 성적 | 일부 과목의 개선을 통해 더 좋은 성적 기대 |
| **-5점 이상** | ⚠️ 성적 하락 주의 | 학습 전략 수정이 필요합니다 |
