# 📚 과목별 성적 분석 대시보드

6개 주요 과목(국어, 수학, 사회, 과학, 영어, 한국사)의 성적을 입력하면 실시간으로 분석하고 시각화해주는 대시보드입니다.

## 🎯 주요 기능

### 1. **성적 입력**
- 6개 과목별 점수 입력 (0~100점)
- 간편한 UI로 빠른 입력

### 2. **실시간 그래프 시각화**
- **레이더 차트**: 6개 과목의 전체적인 성적 분포를 한눈에 볼 수 있음
- **막대 그래프**: 과목별 점수를 비교하기 쉽게 표시

### 3. **주요 통계**
- 전체 평균 점수
- 최고 점수 및 최저 점수
- 점수 편차 (과목 간 격차)

### 4. **상세 분석**
- 📊 **성적 평가**: 현재 성적 수준 분석
- ✅ **강점 과목**: 잘하고 있는 과목 2개
- 🎯 **개선 필요 과목**: 더 공부해야 할 과목 2개
- ⚖️ **균형 분석**: 과목 간 편차 분석 및 조언

## 📖 사용 방법

1. **점수 입력**
   - 각 과목별로 0~100 사이의 점수를 입력합니다
   
2. **분석 버튼 클릭**
   - "📊 점수 분석하기" 버튼을 클릭하면 즉시 분석됩니다
   - 또는 마지막 입력 후 <Enter> 키를 누르면 됩니다

3. **결과 확인**
   - 레이더 차트와 막대 그래프로 점수 분포 확인
   - 통계 정보와 상세 분석 결과 확인

4. **초기화**
   - "🔄 초기화" 버튼으로 모든 입력을 초기화할 수 있습니다

## 🎨 화면 구성

| 섹션 | 설명 |
|------|------|
| 입력 영역 | 6개 과목 점수 입력 |
| 레이더 차트 | 전체 성적 분포 시각화 |
| 막대 그래프 | 과목별 점수 비교 |
| 주요 통계 | 평균, 최고, 최저, 편차 |
| 성적 분석 | 맞춤형 조언 및 피드백 |

## 💡 분석 및 조언

### 성적 평가 기준
- **90점 이상**: 탁월한 성적 🌟
- **80~89점**: 우수한 성적 👍
- **70~79점**: 평범한 성적 📚
- **70점 미만**: 개선 필요 ⚠️

### 편차 분석
- **편차 30점 이상**: 약한 과목 집중 학습 필요
- **편차 30점 미만**: 균형잡힌 성적
