# 📚 성적 추이 분석 대시보드

1학기와 2학기 성적을 입력하고 비교하여 성적 변화를 추적하는 대시보드입니다.

## 🚀 바로 시작하기

https://jongin10.github.io/score-dashboard/ 에 접속하세요!

## 📝 사용 방법

### 1단계: 성적 입력
- **"📝 성적 입력"** 탭에서 시작
- **1학기** 성적 입력 (국어, 수학, 사회, 과학, 영어, 한국사)
- **2학기** 성적 입력
- **"📊 성적 분석하기"** 버튼 클릭

### 2단계: 분석 결과 확인
자동으로 **"📊 성적 분석"** 탭으로 이동하여:
- 📈 **라인 차트**: 두 학기 성적 추이
- 📊 **비교 차트**: 학기 간 과목별 차이
- 📌 **통계**: 평균, 상승도, 최대 상승 과목
- 📋 **비교 테이블**: 과목별 상세 비교
- 💡 **맞춤형 피드백**: 성적 변화 분석 및 조언

## ✨ 주요 기능

✅ **1학기/2학기 성적 비교**  
✅ **실시간 그래프 시각화**  
✅ **과목별 성적 변화도 표시**  
✅ **AI 기반 맞춤형 피드백**  
✅ **모든 기기에서 사용 가능 (반응형)**  

## 💡 피드백 기준

- 🚀 **+5점 이상**: 눈에 띄는 성장!
- 👍 **+1~4점**: 성적 향상
- → **-4~0점**: 안정적인 성적
- ⚠️ **-5점 이상**: 성적 하락 주의
